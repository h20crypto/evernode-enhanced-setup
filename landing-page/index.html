üöÄ Deploying Applications

Basic Deployment Command Structure

evdevkit acquire -i IMAGE:TAG HOST_ADDRESS -m HOURS

IMAGE:TAG - Docker image (e.g., nginx:latest)

HOST_ADDRESS - Evernode host address

HOURS - How long to run (e.g., 24 for 24 hours)

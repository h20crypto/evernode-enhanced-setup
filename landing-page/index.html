⚙️ Installing evdevkit

Step 1: Install Node.js (if not installed)

First, check if you have Node.js installed:

node --version

❌ If you get "command not found", download Node.js from nodejs.org
✅ If you see a version number (v14.0.0 or higher), you're good to go!

Step 2: Install evdevkit globally

npm install -g evdevkit

This installs the Evernode development kit globally on your system.

🛠️ Common Installation Issues

❌ "Permission denied" (Mac/Linux):

sudo npm install -g evdevkit

❌ "npm not found":

Install Node.js from nodejs.org, which includes npm

🔧 Configuration & Setup

Step 1: Generate Deployment Key

Generate a unique key for your deployments:

evdevkit keygen

💾 Important: Copy the privateKey value (starts with 'ed'). You'll need this!

Step 2: Set Environment Variables

🪟 Windows (Command Prompt):

set EV_TENANT_SECRET=sYourXahauSecret
set EV_USER_PRIVATE_KEY=edYourGeneratedKey

🍎 Mac/Linux:

export EV_TENANT_SECRET=sYourXahauSecret
export EV_USER_PRIVATE_KEY=edYourGeneratedKey

🚀 Deploying Applications

Basic Deployment Command Structure

evdevkit acquire -i IMAGE:TAG HOST_ADDRESS -m HOURS

IMAGE:TAG - Docker image (e.g., nginx:latest)

HOST_ADDRESS - Evernode host address

HOURS - How long to run (e.g., 24 for 24 hours)

⚡ Quick Example - Nginx Web Server
evdevkit acquire -i nginx:latest rExampleHostAddress987654321ABC -m 24
📝 WordPress Website
evdevkit acquire -i wordpress:latest rExampleHostAddress987654321DEF -m 48
⚙️ Advanced with Environment Variables
evdevkit acquire -i postgres:13 rExampleHostAddress987654321GHI -m 72 -e POSTGRES_PASSWORD=mypassword

📊 Managing Your Deployments

📋 List Active Instances

evdevkit list

Shows all your running applications and their details

🛑 Stop an Instance

evdevkit terminate INSTANCE_ID

Replace INSTANCE_ID with the ID from 'evdevkit list'

🔄 Monitor Instances

evdevkit status INSTANCE_ID

Check the status of a specific instance using its ID

🛠️ Troubleshooting

❌ "command run not found"

✅ Solution: Use Enhanced Hosts that support Docker web applications

🔧 "evdevkit: command not found"

evdevkit not installed properly

If still failing, try with sudo (Mac/Linux) or run as Administrator (Windows)

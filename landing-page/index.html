📦 Simple Instance Management

Easy 3-step process to create, use, and extend your instances

🚀

1. Create Instance

Deploy your dApp to get a running instance

evdevkit upload contract.zip http://host/
✅ Result: You get a unique instance ID and URL
⚡

2. Use Instance

Access your dApp through the provided URL

https://host/instance-abc123
💡 Tip: Your dApp runs 24/7 on the host
🔄

3. Extend Instance

Keep your dApp running longer

evdevkit extend [instance-id] 30d
⏰ Default: Instances run for 30 days

👤 Single-Use Tenant Guide

Complete setup guide for deploying your first dApp

📦

Step 1: Install evdevkit

npm install -g evdevkit

Install the Evernode development toolkit globally on your system

🔑

Step 2: Generate Keys

evdevkit keygen

Generate your tenant keys for secure dApp deployment

🚀

Step 3: Deploy dApp

evdevkit upload contract.zip http://your-host-ip/

Upload your contract to an enhanced Evernode host

📊

Step 4: Monitor & Manage

evdevkit status [instance-id]

Monitor your dApp's performance and health

💻 Terminal Commands Quick Reference

Windows (CMD/PowerShell)

# Install Node.js first from nodejs.org
npm install -g evdevkit
evdevkit keygen
evdevkit upload contract.zip http://host-ip/

macOS/Linux (Terminal)

# Install Node.js with package manager
sudo npm install -g evdevkit
evdevkit keygen
evdevkit upload contract.zip http://host-ip/
